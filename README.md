# 🌍 AI Smart Translator (CodeAlpha Task 1)

A modern AI-powered language translation tool built using FastAPI.

## Features
- Multi-language translation
- Auto language detection
- Text-to-Speech
- Copy to clipboard
- Dark mode
- AJAX-based fast UI

## Tech Stack
- FastAPI (High-performance backend framework)
- HTML5 & CSS3 (Responsive UI design)
- JavaScript (Fetch API for async requests)
- External APIs (LibreTranslate & Google Translate fallback)

## Run
uvicorn main:app --reload
