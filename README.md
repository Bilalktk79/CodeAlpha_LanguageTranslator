# 🌍 AI Smart Translator (CodeAlpha Task 1)

A modern AI-powered language translation tool built using FastAPI.

## Features
- Multi-language translation
- Auto language detection
- Text-to-Speech
- Copy to clipboard
- Dark mode
- AJAX-based fast UI

## Tech Stack
- FastAPI
- JavaScript (Fetch API)
- Bootstrap

## Run
uvicorn main:app --reload
